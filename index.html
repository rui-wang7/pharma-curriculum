Course Ⅰ · Overview
药物创新的逻辑（概览）
一门"操作系统"课：一个药是怎么从疾病生物学一路走到市场的。六个分步：像药物猎手一样思考 → 创新史 → 弹药库 → 主战场 → 当下热点 → 看清未来。学完能看懂任何一个 biotech 项目在赌什么。
6 个分步 · 已完成
Course Ⅱ · Immunology
免疫学入门
跟着经典教材 Janeway《免疫生物学》的框架，从零讲清"免疫系统是身体的国防部"：先天免疫 → 适应性免疫 → 识别 → 效应 → 记忆/疫苗 → 免疫失调 → 免疫药物。全程大白话 + 军事/生活类比。
