免疫炎症词典 Immunology

元规则：几条"细胞因子"在多个自免病里反复作乱。打法两大类——中和细胞因子/受体（抗体，安全但要打针）或阻断胞内信号（小分子，口服但更易全身影响）。同一靶点常横跨多病，价值随适应症滚雪球（pipeline-in-a-product）。

读法

已成功 有获批药 · 爬坡中 有信号/新机制 · 反复失败 屡战屡败。留意每条的"适应症地图"——横向扩病是本战场最大价值来源。

A · 细胞因子中和：抗体的主场

把作乱的可溶信号分子（或其受体）用抗体"抓走"。原则：打得越上游，覆盖病人越广，但也越可能过度抑制免疫。

类风关/银屑病/IBD · TNF-α
已成功 · 元祖
核心促炎因子 → TNF-α → 抗体 / 融合蛋白
代表药&公司adalimumab/Humira（AbbVie）——曾是全球药王；etanercept、infliximab
解法 / 解毒剂"一药多病"原型：类风关→银屑病→强直→克罗恩→溃结一路扩
pipeline-in-a-product 的鼻祖，也是专利悬崖的教材：生物类似药一上市收入断崖，逼 AbbVie 用 IL-23/JAK 接棒。
银屑病/IBD · IL-23 (p19)
已成功 · 当红
Th17 上游驱动 → IL-23 → 抗体
代表药&公司risankizumab/Skyrizi（AbbVie）、guselkumab（J&J）；上一代 ustekinumab 打 IL-12/23 p40
Humira 的接班人。"打更上游"往往=更干净更持久，是免疫里 best-in-class 接力的样板。皮肤清除率成军备竞赛。
银屑病/关节炎 · IL-17A/F
已成功
Th17 效应因子 → IL-17 → 抗体
代表药&公司secukinumab（Novartis）、ixekizumab（Lilly）、bimekizumab（双靶 A+F，UCB）
与 IL-23 同场竞争皮肤清除；但在 IBD 上 IL-17 反而无效甚至加重——提醒"同通路不同病，结果可能相反"。
特应性皮炎/哮喘/COPD · IL-4Rα (IL-4/13)
已成功 · 巨头
2 型炎症主轴 → IL-4/IL-13 共用受体 → 抗体
代表药&公司dupilumab/Dupixent（Regeneron/Sanofi），已扩到 COPD、嗜酸食管炎、结节性痒疹、慢性鼻窦炎等
"2 型炎症"通吃者，适应症地图最宽的免疫药之一——每个新适应症都是一次重估。单靶点撑起数百亿销售。
嗜酸性哮喘/EGPA · IL-5 / IL-5Rα
已成功 · 生标选人
嗜酸性粒细胞驱动 → IL-5 → 抗体
代表药&公司mepolizumab（GSK）、benralizumab（AZ，直接清嗜酸细胞）
"用血嗜酸细胞计数选人"的精准免疫：把哮喘按生物标志物分层，只治真正 2 型高炎症的那群。
类风关/CRS/巨细胞动脉炎 · IL-6 / IL-6R
已成功 · 兼职解毒剂
全身炎症放大器 → IL-6 → 抗体
代表药&公司tocilizumab（罗氏）、sarilumab（Regeneron/Sanofi）
解法 / 解毒剂同一药是 CAR-T 细胞因子风暴（CRS）的标准解毒剂——肿瘤科反向借用免疫科的药
一个通路两个战场：治自免，也救 CAR-T 的命。跨领域"老药新用"的典型。
自身炎症/痛风/心血管 · IL-1β
已成功
先天免疫警报 → IL-1β → 抗体 / 受体拮抗剂
代表药&公司canakinumab（Novartis）、anakinra；用于罕见自身炎症综合征、难治痛风
CANTOS 试验曾证明抗 IL-1β 能降心血管事件——"炎症假说"治动脉粥样硬化的里程碑（虽因性价比未转化为大适应症）。
重症哮喘 · TSLP
已成功 · 最上游
上皮"警报素"（炎症瀑布顶端）→ TSLP → 抗体
代表药&公司tezepelumab（Amgen/AZ）——不分表型都有效
越往上游打覆盖越广：TSLP 在最顶端，所以对多种哮喘表型通用。上游打法的代表。
溃结/克罗恩 · TL1A
爬坡中 · 大交易
兼抗炎+抗纤维化 → TL1A → 抗体
代表药&公司tulisokibart（默沙东，收购 Prometheus）、duvakitug（赛诺菲/Teva）
解法 / 解毒剂卖点是"带生物标志物选人"——想先挑出会响应的病人再治
当下 IBD 最热的新机制，大药企砸重金抢购的下一代免疫资产。看它能否兑现"抗纤维化"这个 IBD 长期痛点。

B · 胞内信号阻断：小分子的立足点

抗体进不了细胞，胞内信号（激酶、受体下游）就是小分子的天下。优势是口服便宜，代价是选择性不够时容易"广谱抑制"惹副作用。

多种自免 · JAK 1/2/3
已成功 · 带黑框
众多细胞因子共用的胞内枢纽 → JAK-STAT → 口服小分子
代表药&公司tofacitinib（Pfizer）、upadacitinib（AbbVie）、baricitinib；FDA 黑框（心血管/血栓/感染）
口服便利 vs 抗体安全的经典权衡。黑框是"广谱阻断共用枢纽"的代价，差异化只能靠更高选择性——引出 TYK2。
银屑病等 · TYK2（变构）
已成功 · 更安全
JAK 家族一员，但走变构口袋 → TYK2 假激酶域 → 口服小分子
可成药性不抢 ATP 口袋、而抓调控域——换来更干净的选择性
代表药&公司deucravacitinib/Sotyktu（BMS），无 JAK 类黑框
"同一家族、换个结合位点=更好安全性"的漂亮范例。变构抑制是新一代小分子提高选择性的通用招法。
溃结/多发性硬化 · S1P 受体
已成功
淋巴细胞出淋巴结的"闸门" → S1P1 → 口服小分子
代表药&公司fingolimod/ozanimod（BMS）、etrasimod（辉瑞，溃结）；MS 用 siponimod
机制巧妙——"把免疫细胞关在淋巴结里出不来"。口服、免打针，是 MS 和 UC 的重要口服选项。
银屑病/银屑病关节炎 · PDE4
已成功 · 温和
胞内 cAMP 降解酶 → PDE4 → 口服/外用小分子
代表药&公司apremilast（Amgen 口服）、roflumilast（外用）
疗效温和但安全性好、无需化验监测——占据"轻中度、怕打针怕黑框"的病人层。安全性也是一种差异化。

C · 细胞与受体：组织选择与清除 B 细胞

溃结/克罗恩 · 整合素 α4β7
已成功 · 肠道选择性
淋巴细胞归巢肠道 → α4β7 → 抗体
代表药&公司vedolizumab（武田）——只拦肠道免疫、全身少受累
"组织选择性"是降副作用的聪明思路：不全身压制，只在犯事器官动手。安全性溢价明显。
系统性红斑狼疮 · BAFF/BLyS
已成功 · 狼疮少有
B 细胞存活因子 → BAFF → 抗体
代表药&公司belimumab（GSK）——狼疮几十年里罕见的获批新药之一
狼疮是出了名的"临床坟场"（异质性高、终点难）。BAFF 的成功来之不易，凸显选人和终点设计之难。
类风关/多发性硬化 · CD20（B 细胞）
已成功 · 跨界
清除 B 细胞谱系 → CD20 → 抗体
代表药&公司rituximab（罗氏）；MS 用 ocrelizumab、ofatumumab
同一个肿瘤靶点（见肿瘤词典）在自免里再立功——清 B 细胞治 MS 的成功，反过来点燃了"CAR-T 治自免"的想象。
慢性荨麻疹/AD 瘙痒 · IL-31 · BTK(免疫型)
爬坡中
瘙痒/肥大细胞信号 → IL-31 或 BTK → 抗体 / 口服小分子
代表药&公司nemolizumab（IL-31，结节性痒疹已获批）；remibrutinib（诺华 BTK，CSU/MS 在研）
把"痒"本身当靶点——IL-31 是瘙痒专线；免疫型 BTK 抑制则想复制肿瘤 BTK 的成功、去掉出血副作用。

D · 前沿：把肿瘤的"活药"搬来治自免

重症狼疮/肌炎/硬皮病 · CD19（CAR-T / 双抗）
爬坡中 · 最热
深度清除自身反应性 B 细胞 → CD19 → CAR-T / T 细胞衔接双抗
因果强度早期病例"无药停药缓解"信号惊人，样本仍小
代表药&公司德国 Schett 团队开创；诸多 biotech（含中国）跟进 CAR-T 与 CD3×CD19/CD20 双抗
解法 / 解毒剂"免疫系统重启"假说：清干净 B 细胞后再生成的是不致病的；CRS 仍用 tocilizumab 兜底
当下免疫学最激动人心的外延——把肿瘤的细胞治疗搬进自免。若成，重塑整个自免治疗范式；风险在长期安全与可及性/成本。
机制通用 · IgG 回收 FcRn
已成功 · 平台
降低致病性自身抗体 → FcRn（IgG 回收受体）→ 抗体 / Fc 片段
代表药&公司efgartigimod（argenx）、rozanolixizumab；用于重症肌无力、CIDP、ITP 等
"抗体介导病"的通用开关：不管哪种自身抗体，都靠加速清除 IgG 一网打尽——一个机制横扫一类疾病的平台逻辑。
这一页的元规则

看免疫项目三问：① 打在通路哪一层（越上游越通用，也越可能过度抑制）；② 适应症地图能扩多宽（一靶跨几大病，价值翻几倍）；③ 安全性差异化（TYK2 之于 JAK、组织选择性之于全身抑制——在免疫里"更安全"本身就能赢）。下一波盯 TL1A 类"带选人标志物的新机制"和 CD19 细胞治疗治自免的兑现。
