神经词典 CNS

被称为"坟场"：血脑屏障挡药、动物脑不像人脑、终点慢到要熬几年花几亿。但正因难，突围格外值钱。规律很清楚——有明确遗传学撑腰的病翻身（SMA、CGRP、SOD1），靠"假说猜"的病继续折戟（tau、α-syn）。而精神领域正迎来几十年一遇的新机制。

读法

本页把"翻身仗"和"坟场"并排放，看同一战场里因果强度如何决定成败。已成功／爬坡中／反复失败。

A · 神经退行：遗传学决定成败

脊髓性肌萎缩 SMA · SMN2 / SMN1
已成功 · 翻身仗
单基因缺陷 → 补足 SMN 蛋白 → ASO / 口服剪接 / 基因治疗
因果强度单基因病，因果满格
代表药&公司nusinersen（Biogen ASO 鞘内）、risdiplam（罗氏口服剪接）、onasemnogene（诺华 AAV 一次性）
CNS 最漂亮的胜利：一个致死婴儿病，被三种模态从不同角度攻克。原因只一个——因果干净。
阿尔茨海默 · 淀粉样蛋白 Aβ
爬坡中 · 争议缠身
淀粉样斑块假说 → Aβ → 抗体
因果强度几十年屡败后近年才勉强验证部分获益
代表药&公司lecanemab/Leqembi（卫材/Biogen）、donanemab/Kisunla（Lilly）——延缓有限、有脑水肿/出血(ARIA)风险
最贵的"半胜"：证明清斑块能小幅延缓，但获益幅度与安全性争议不断。CNS"慢而模糊终点"的代价样本。
阿尔茨海默/帕金森 · tau · α-突触核蛋白
反复失败 · 未攻克
神经退行"第二假说" → tau / α-syn → 抗体 / ASO 多次落空
因果强度相关性强、干预获益未证
代表药&公司抗 α-syn（cinpanemab/prasinezumab）帕金森试验失败；抗 tau 多数未达标
"病理相关≠致病可干预"。这些靶点上因果这把尺子还没给到及格分——继续烧钱验证假说。
亨廷顿病 · HTT（亨廷顿蛋白）
爬坡中 · 起伏
单基因显性遗传 → 降低 HTT → ASO / 基因治疗
因果强度单基因、因果极清楚
代表药&公司tominersen（罗氏，一度失败后调剂量再试）；uniQure AMT-130 基因治疗现积极信号
因果满分却也翻车——说明 CNS 即便靶点对，"降多少、何时降、怎么送进脑"本身就是成败关键。
ALS（渐冻症）· SOD1 · C9orf72
爬坡中 · 亚型突破
遗传性 ALS 突变 → SOD1 / C9orf72 → ASO 鞘内
因果强度遗传亚型因果明确（各占少数）
代表药&公司tofersen（Biogen，靠生物标志物 NfL 加速批准）；C9orf72 ASO 多在研（有失败）
在无解的 ALS 里，先啃"有明确基因"的小亚型——典型"用遗传学对冲因果风险"打法，并示范 NfL 作为替代终点。
多发性硬化 · CD20 · BTK · S1P
已成功 + 前沿
神经免疫（B 细胞/信号）→ CD20/BTK/S1P → 抗体 / 口服小分子
代表药&公司ocrelizumab（CD20，罗氏）已确立；BTK 抑制剂（tolebrutinib/fenebrutinib）攻"进展型 MS"这块硬骨头
MS 与免疫词典交叉。看点是能否打进"非炎症性的进展型 MS"——那是当前最大未满足需求，BTK 是当红赌注。

B · 疼痛与偏头痛：绕开血脑屏障的胜利

共同智慧：把作用点放在外周神经，既避开血脑屏障难题、又躲开中枢副作用（成瘾、镇静）。

偏头痛 · CGRP
已成功 · 大众市场
偏头痛核心介质 → CGRP / 受体 → 抗体（预防）+ 口服小分子（急性）
代表药&公司erenumab（Amgen/诺华）等抗体；gepant 类口服 rimegepant/atogepant（Pfizer/AbbVie）
证明 CNS 也能有"大众化、可测、可预防"的成功。作用多在外周血管/三叉神经，巧妙绕开血脑屏障。
急/慢性疼痛 · Nav1.8（钠通道）
已成功 · 非阿片新类
外周痛觉信号 → Nav1.8（外周特异）→ 口服小分子
因果强度人类遗传学背书（SCN 突变改变痛觉）
代表药&公司suzetrigine/Journavx（Vertex，2025 获批急性痛）
数十年来第一个新机制止痛药类别，且不成瘾——对阿片危机是标志性事件。"选外周亚型通道绕开中枢"的范式。

C · 精神疾病：几十年一遇的新机制潮

精神药几十年停在"多巴胺/血清素"老机制上。近两年终于出现真正的新作用机制，重估了整个领域。

精神分裂症 · 毒蕈碱 M1/M4
已成功 · 新机制
绕开多巴胺 D2 → 胆碱能 M1/M4 → 口服小分子
代表药&公司xanomeline-trospium/Cobenfy（原 KarXT，BMS 收购 Karuna，2024 获批）
解法 / 解毒剂配一个"不进脑"的外周拮抗剂（trospium）来抵消胆碱能副作用——巧妙的组合工程
70 年来第一个非 D2 机制的抗精神病药。证明精神科仍有全新靶点可挖，点燃整个 CNS 精神管线的重估。
抑郁/产后抑郁 · NMDA · GABA-A（速效）
已成功 · 速效
谷氨酸/GABA 系统 → NMDA / 神经甾体 → 鼻喷 / 口服
代表药&公司esketamine/Spravato（强生，鼻喷）；zuranolone/Zurzuvae（2023，产后抑郁口服短疗程）
打破"抗抑郁药要吃几周才起效"的旧范式——数小时到数天见效。机制新颖也带来滥用/镇静的管理挑战。
难治抑郁/PTSD · 5-HT2A（迷幻药）
爬坡中 · 监管坎坷
血清素 2A 受体 → psilocybin / MDMA 辅助治疗 → 小分子 + 心理治疗
因果强度早期信号强，试验设盲/主观终点难
代表药&公司psilocybin（Compass）在研；MDMA（Lykos）2024 被 FDA 拒（CRL）
机制热但监管难：疗法离不开心理治疗、无法双盲、安全与滥用顾虑重。提醒"科学信号"要过"试验可证性"这关。

D · 递送：把药真正送进脑

CNS 的永恒瓶颈是血脑屏障。与其换靶点，不如解决"怎么进去"——这本身就是一个平台级机会。

机制通用 · 转铁蛋白受体 TfR（分子木马）
爬坡中 · 平台
搭"过屏障的顺风车" → TfR 胞吞转运 → 双特异抗体 / 融合"脑穿梭"
代表药&公司Denali "TV" 平台（酶替代进脑治溶酶体病）、Roche brain shuttle；把大分子"背"过血脑屏障
不改靶点、改递送——一旦成熟，能把一整批"打得着但送不进"的 CNS 靶点解锁。典型的"价值向平台迁移"。
癫痫 · SV2A · 离子通道
已成功 · 成熟基石
神经元过度放电 → SV2A / 钠钙通道 → 口服小分子
代表药&公司levetiracetam（SV2A）等；罕见癫痫用 cannabidiol、fenfluramine
成熟对症领域：症状可测、药能进脑，是 CNS 里"正常运转"的部分。创新已转向难治性/遗传性癫痫亚型。
这一页的元规则

CNS 胜负几乎由两件事决定：① 有没有硬遗传学（有→SMA/HTT/SOD1；无→tau/α-syn 长期折戟）；② 能否绕开或穿过血脑屏障 + 有没有快读数（CGRP/Nav1.8 走外周、SOD1 用 NfL、TfR 平台专攻递送）。精神领域则刚打开新机制窗口（M1/M4、速效抗抑郁）。看 CNS 项目，先数它占了上面几条。
