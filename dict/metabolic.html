代谢词典 Metabolic

这个战场的独特优势：因果强度极高——很多靶点有"天生带某基因变异的人"当活体实验，等于照抄人体答案；而且读数快（血脂/血糖/体重可测），临床风险相对低。当下最大的 TAM 故事（GLP-1）就在这。

读法

特别看每条的"因果强度"四格——本页多个靶点满格，因为有人类遗传学直接背书，这正是它们成功率高的根本。已成功／爬坡中／反复失败。

A · 血脂与动脉粥样硬化

逻辑统一：降低致动脉粥样硬化的脂蛋白（LDL、Lp(a)、富含甘油三酯脂蛋白）以减少心梗中风。这里是"人类遗传学指路"最密集的地方。

高胆固醇/心血管 · PCSK9
已成功 · 满分范本
调控 LDL 受体降解 → PCSK9 → 抗体 & siRNA
因果强度天生失活者终身低 LDL、少心梗且健康
代表药&公司抗体 evolocumab（Amgen）/alirocumab（Regeneron，几周一针）；siRNA inclisiran（Novartis，半年一针）
解法 / 解毒剂"同靶点换更持久弹药"：抗体→siRNA→在研口服小分子（enlicitide）/基因编辑（VERVE，一次性）
"因果强度"满分案例，也是"老靶点+新模态=新周期"最干净的示范。记死它。
严重高甘油三酯 · ApoC3
已成功 · 刚扩
甘油三酯代谢刹车 → ApoC3 → ASO / siRNA
因果强度天然缺失者甘油三酯低、心血管更好
代表药&公司olezarsen/Tryngolza（Ionis，2024 获批 FCS）；plozasiran（Arrowhead）在研
又一"人类遗传学指路"胜利。RNA 药在肝脏代谢靶点上的又一城；下一步向更大的高甘油三酯人群扩。
纯合子家族性高胆固醇 · ANGPTL3
已成功 · 罕见
脂代谢调节 → ANGPTL3 → 抗体 / siRNA
代表药&公司evinacumab（Regeneron 抗体）；zodasiran 在研
对连 LDL 受体都没有的极端病人也有效——因为走了不依赖 LDL 受体的旁路。示范"选对机制能绕开病理死路"。
心血管残余风险 · Lp(a)
爬坡中 · 结局待读
遗传决定的独立危险因素 → Lp(a) → ASO / siRNA / 口服
因果强度遗传学强证据，临床结局待验
代表药&公司pelacarsen（诺华 ASO）、olpasiran（Amgen siRNA）、lepodisiran（Lilly）、muvalaplin（口服）——III 期结局试验中
经典"遗传学满分、就等结局试验"的赌局：能大幅降 Lp(a)已确定，能否降心梗是几十亿美元的悬念。
他汀不耐受降 LDL · ATP 柠檬酸裂解酶 (ACL)
已成功 · 补位
胆固醇合成上游 → ACL（肝特异）→ 口服小分子（前药）
代表药&公司bempedoic acid/Nexletol（Esperion）——前药只在肝激活，避开肌肉副作用
"前药+组织特异激活"绕开他汀的肌痛痛点，占住"不耐受他汀"的补位市场。设计巧思胜过机制新颖。
升 HDL 降风险 · CETP
反复失败 · 局部复活
"好胆固醇"假说 → CETP → 口服小分子
因果强度升 HDL 无临床获益，但降 LDL/ApoB 或有
代表药&公司torcetrapib/dalcetrapib/evacetrapib 全败；obicetrapib（NewAmsterdam）改押降 LDL 重启 III 期
"升 HDL 假说"崩塌的教科书：相关性≠因果。CETP 若翻身，靠的是降 LDL 而非当初的升 HDL 故事——机制叙事会变。

B · 肥胖与糖尿病：当下最大的超级周期

incretin（肠促胰素）生物学把"减重"从美容变成能改善心/肾/肝硬结局的大适应症。战场已从"有没有效"转到"更强减重 + 口服 + 保肌肉 + 支付"。

肥胖/2型糖尿病 · GLP-1 (/GIP)
已成功 · 超级周期
肠促胰素信号 → GLP-1 受体（±GIP）→ 多肽 / 口服小分子
因果强度减重+心血管获益 III 期确证
代表药&公司semaglutide/Wegovy（诺和诺德）、tirzepatide/Zepbound（礼来，双靶 GIP/GLP-1）
解法 / 解毒剂前沿：口服 orforglipron、三靶 retatrutide、月制剂 MariTide；扩到 MASH/肾/睡眠呼吸暂停/骨关节炎
十年一遇的 TAM 级机会：潜在人群以亿计、适应症滚雪球。瓶颈从"疗效"转到"产能、口服、支付、长期安全"。
肥胖前沿 · Amylin
爬坡中 · 组合拳
饱腹/胃排空信号 → amylin 受体 → 多肽（常与 GLP-1 联用）
代表药&公司cagrilintide、CagriSema（诺和诺德）；petrelintide（勃林格/Zealand）
GLP-1 之后减重的"下一层加法"。看点是"叠加能否在保肌肉/耐受性上胜过单纯加大 GLP-1 剂量"。
2型糖尿病/减重 · 胰高糖素受体 (三靶)
爬坡中 · 最强减重
增能量消耗+降食欲 → GLP-1×GIP×GCGR 三激动 → 多肽
代表药&公司retatrutide（礼来）早期减重数据逼近减重手术量级
"加靶点=加疗效"的极致。悬念在长期安全与保肌肉——减得多不等于减得健康。
糖尿病/心衰/慢性肾病 · SGLT2
已成功 · 意外扩张
肾脏葡萄糖重吸收 → SGLT2 → 口服小分子
代表药&公司dapagliflozin（AZ）、empagliflozin（BI/Lilly）
本为降糖药，却在心衰和慢性肾病拿到大适应症——"意外的心肾获益"是心血管代谢的常见惊喜，留意 label 外延。
2型糖尿病 · DPP-4 / 胰岛素
已成功 · 基石
血糖控制基础层 → DPP-4（保住内源 GLP-1）/ 胰岛素替代 → 口服 / 注射
代表药&公司sitagliptin（默沙东 DPP-4）；胰岛素百年基石，前沿是周制剂 icodec 与智能闭环
成熟基石类：增长慢、竞争充分，但奠定"血糖可测=读数快"的战场底色。创新点已上移到 incretin 和周制剂。

C · 代谢相关脂肪肝 MASH：刚破冰的大战场

MASH（原 NASH）是"没有终点药、患者以千万计"的老大难。2024 年终于破冰，也成为 GLP-1 外延的重要下一站。

MASH（脂肪肝炎）· 甲状腺激素受体 THR-β
已成功 · 首个破冰
肝脏脂代谢 → THR-β（肝特异）→ 口服小分子
因果强度组织学改善已 III 期达标
代表药&公司resmetirom/Rezdiffra（Madrigal，2024 首个获批 MASH 药）
解法 / 解毒剂"肝脏靶向的甲状腺激素"——只在肝激活、避开全身甲亢副作用
一个空白大战场被打开的信号事件。但真正的降门槛可能来自 GLP-1/FGF21——MASH 未来是多机制联用的混战。
MASH · FGF21 类似物
爬坡中 · 抗纤维化
代谢+抗纤维化 → FGF21 通路 → 工程化多肽
代表药&公司efruxifermin（Akero）、pegozafermin（89bio）——纤维化逆转数据受关注
MASH 的关键不只是消脂肪，更是逆转纤维化。FGF21 类若能证明抗纤维化，将与 THR-β/GLP-1 形成联用底座。
这一页的元规则

代谢是"因果强度"最友好的战场：先问有没有人类遗传学背书——有，成功率高一截（PCSK9/ApoC3/ANGPTL3）；警惕相关性冒充因果（CETP 升 HDL、CNS 式假说在这也翻过车）。其次看弹药持久度（抗体几周→siRNA 半年→基因编辑一次性，越持久越贴"预防"）。GLP-1 则把战场从"控指标"抬升到"改结局 + 超大人群 + 多器官外延"。
